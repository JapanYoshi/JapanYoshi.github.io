This section will hide when viewed from your streaming software.

To test this out, simply connect your controller to your computer, and if it is already connected, press any button or tilt any stick. This can only detect the joystick with ID #0, which means if the right controller is not detected, you should disconnect all controllers, then reconnect your desired one.

This overlay is compatible with Haley's Xbox One controller, and is expected to be compatible with other major controllers.

To use this in your stream, simply add this page as a Browser Source with these settings:

  • Width: 512px
  • Height: 320px
  • Use custom framerate: you decide, I don't even know what this does
  • Custom CSS: Default should be enough. In fact, you probably don't need any custom CSS.
