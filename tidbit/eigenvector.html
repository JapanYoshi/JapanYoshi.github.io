Eigenvector

multi-mode noteskin for Stepmania 5.0 ≤ v < 5.2 and Etterna

concept

Eigenvector is a noteskin that is compatible with any mode that uses the 9 main keys (center, cardinal directions, and diagonal directions). It provides a futuristic look and feel, which makes different modes look cohesive with each other.

name

ei·gen·vec·tor /ˈaɪ·gənˌvɛk·tɚ/

(n.) a vector that points in the same direction after a linear transformation

The noteskin was named after this because of its vision to provide a direction that stays between modes.

how to use

Eigenvector is made of common files and mode-specific "interfaces".

By extracting the contents of the archive to the Noteskins folder (as opposed to the folder for one mode only), you will be performing these steps.

  1. Extract assets to /Noteskins/common/Eigen.This contains data files for all modes.
  2. Extract metrics.ini to /Noteskins/[mode]/Eigenvector. This will let the specific mode access the data files.
  3. If Stepmania is running, reload metrics and textures using F2. If not, simply start Stepmania.

That is to say, the Noteskins folder should look like this:

  • Noteskins/
    • common/
      • Eigen/
        • [data files]
    • dance/ (for example)
      • Eigenvector/
        • metrics.ini
