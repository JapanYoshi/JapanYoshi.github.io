このノートスキンについて

STARLiGHT （ スターライト ）  は、Stepmania の dance モード用のノートスキンです。Solo スタイルにも対応しています。『DDR XX （ エックスエックス ） - STARLiGHT （ スターライト ） -』というプロジェクトの一環として作られました。

背景が明るくても暗くても見えやすい、モダンなデザインは、懐かしのダンレボの矢印を2020年代に蘇らせます。

バリエーションはなんと7通りあります。詳しくは次のセクションで。

免責事項と使用規約

このノートスキンは次の用途で使えます。

  • Stepmania のプレイ・譜面動画を作るため
  • 音ゲー関連のコラ画像を作るため
  • 参加費が無料か、募金活動のための Stepmania の大会を開催・宣伝するため

このノートスキンを次の用途で使用することを禁じます。

  • ノートスキンの著作権者である、デザイナー（はるるちゃん）と DDR XX プロジェクトをクレジット表示せず、再転載するため
  • 商用的な Stepmania の大会を開催・宣伝するため
  • 公式なDDRの大会を宣伝するため

他の用途については、適宜に計らってください。自信が無い場合、デザイナーに連絡してください。

8分 = 青

24分 = 紫
32分 = 黄色
48分 = 紫

STARLIGHT-RAINBOW

DDR 公式の RAINBOW 準拠。

拍ちょうど = オレンジ
¼拍以前 = ピンク
½拍以前 = 青
¾拍以前 = 紫

STARLIGHT-CYCLE

DDR 公式の RAINBOW を12色に拡張したもの。

拍ちょうど = 赤
1/6～1/4拍 = 黄色
5/12～1/2拍 = 水色
2/3～3/4拍 = 紫
他 = 中間

STARLIGHT-VIVID

DDR 公式の VIVID 準拠。ノーツの色は虹色に循環するが、タイミングによって位相が変わる。

拍ちょうど = オレンジから
¼拍以前 = ピンクから
½拍以前 = 青から
¾拍以前 = 紫から
その後 = オレンジから

STARLIGHT-FLAT

DDR 公式の FLAT 準拠。ノーツの色はタイミングにかかわらず同じペースで虹色に循環する。

使い方

STARLiGHT の7通りのバリエーションは、共通するアセットを使っています。STARLIGHT-NOTE と STARLIGHT-CYCLE 以外のバリエーションを使うためには、STARLIGHT-NOTE が必須です。

インストールするには、次の手順をたどってください。

  1. 全てのフォルダーを/Noteskins/danceへ解凍する。（Stepmania 5.3 を使用している場合、/Appearance/Noteskins/danceに解凍）
  2. Stepmania が今実行されている場合、F2を押してメトリックとテクスチャーを再読み込みする。実行されていない場合、そのままStepmaniaを起動する。

インストールが終了した後、あなたのNoteskinsフォルダは次のようになっているはずです。

  • Noteskins/
    • dance/
      • starlight-note/（必須）
        • metrics.ini
        • Noteskin.lua
        • 共通アセット
      • starlight-cel/（例として）
        • metrics.ini
        • バリエーション独自のアセット
      • 他のノートスキン...
